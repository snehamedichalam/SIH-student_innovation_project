# SIH-student_innovation_project
CitizenOne – Control Room Dashboard
CitizenOne is a government service monitoring dashboard UI designed to help authorities track queues, crowd levels, and service performance in real time.

Built as a prototype under the theme
Smart Automation & Digital Inclusion – Barriers to Government Services.

🚀 Features
📊 KPI metrics (queue, wait time, appointments, completion rate)
📈 Live footfall analytics
🔮 Crowd forecast visualization
🍩 Service distribution chart
🗺️ Office crowd map (Low / Medium / High)
🏢 Queue by office table
🚨 Live alert system
🛠 Tech Stack
HTML
CSS
JavaScript
SVG Charts
🎯 Goal

To demonstrate how a control-room style dashboard can improve decision-making and reduce waiting time in government service centers.
